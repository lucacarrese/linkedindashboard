# Design Brief — test-mock

## Post Content
```
5 Claude Code agents replaced my entire GTM stack.

Each one runs 24/7. Zero manual input.

→ Prospect Research Agent
   ↳ Pulls ICP companies from LinkedIn + Apollo

→ Enrichment Agent
   ↳ Scores leads Tier 1/2/3 via Clay

→ Sequence Agent
   ↳ Writes personalized emails per tier

→ CRM Agent
   ↳ Updates HubSpot + deduplicates automatically

→ Reporting Agent
   ↳ Sends weekly Slack digest with pipeline stats

The stack cost $600/month before.
Now it costs $20/month in Claude credits.

What's the first agent you'd build?
```

## Hook (first line)
> 5 Claude Code agents replaced my entire GTM stack.

## Tone
Bold / Authority

## Layout Type
List Card (Layout A — toggle rows)

## Key Visual Message
5 agents, each doing a real job, running autonomously — the whole GTM stack replaced

## Dominant Element
The number 5 and the $600 → $20 cost reduction

## Color Mood
dark

## Notes for Designer
Show each agent as a toggle row inside one main card. Make "$600/month → $20/month" stand out visually as a stat strip below the cards.
