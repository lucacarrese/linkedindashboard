# Example 06 — Budget Tier Breakdown
**Format:** Contrarian Hook → Context → Numbered Tiers with Tools per Budget
**Hook type:** Objection crusher ("not having budget is not an excuse")
**CTA type:** Results question

---

Not having a budget to run outbound should not be an excuse anymore.

Here are tools for every budget:

Every day, the outbound game steps up, so should your tool stack.

From experience, companies that do not invest in the outbound stack are not getting results from outbound.

This is my list:

1️⃣ Under $300/mo
→ Instantly.ai ($37+) or lemlist ($69+)

↳ List building + outreach, all in one place
↳ Simple setup

2️⃣ $300–$550/mo
→ Add Clay ($149+)
→ Add Expandi.io for Linkedin outreach ($99)

↳ Keep Instantly or lemlist
↳ Layer in better data.
↳ Pull from new data sources, scrape more leads
↳ Quality data = higher reply rates

3️⃣ $550–$1,600/mo
→ Build a focused stack.

↳ Apollo for B2B data (~$99/mo)
↳ FullEnrich, Prospeo.io, LoneScale, for enrichment (~$100/mo)
↳ Vector for website visitors (~$0-149/mo)
↳ Clay Pro Plan (~$800/mo)
↳ Attio CRM ($50-250/mo)

4️⃣ $1,600–$5,500+/mo
→ Intent: Common Room (~$999/mo)
→ Copywriting and strategy: Twain starting at ($35/m)

↳ Everything above, fully automated

I've seen teams win deals with $100/mo and teams waste $3,000/mo with no results.

The stack matters more than the spend.
