# Example 09 — Step-by-Step System (Volume Credibility Hook)
**Format:** Volume Proof Hook → 6 Numbered Steps with Bold Titles + → Bullets → Tech Stack → Summary List
**Hook type:** Volume credibility + promise ("1M emails so you don't have to")
**CTA type:** Soft visual CTA (not aggressive)

---

We sent 1,000,000+ cold emails so you don't have to.
Here is the 6-step system we deploy across 70+ clients.

1️⃣ LEAD SOURCING

Your campaign is only as strong as your list.

We pull from 6+ sources:
→ Apollo.io + LeadMagic for broad targeting
→ Sales Navigator + Prospeo.io for roles
→ Ocean.io / DiscoLike for lookalikes
→ Clay for enrichment + filtering
→ Openmart for local businesses

One source only = reply-rate killer.

2️⃣ CAMPAIGN STRATEGY

Most teams spray random lists and pray.

We run trigger-based campaigns:
→ New hires
→ Funding rounds
→ Job postings
→ Tech stack changes

Relevance gives 3x higher reply rates.
We set 3-5 evergreen campaigns per client that update daily and run on autopilot.

3️⃣ COPYWRITING

What works after 10,000,000+ emails:

→ Line 1: Trigger or observation
→ Lines 2-3: How you solve it
→ Line 4: Open-ended question
→ P.S: Personalized touch (can be used as intro)

Lead with the trigger, not with your name and title.

4️⃣ CLAY WORKFLOW

Where manual work dies.

→ Input raw leads
→ Enrich company + contact data
→ Score: GPT tiers 1, 2, 3
→ Route by tier across email, calls, LinkedIn
→ Sync everything to CRM

5️⃣ EVERGREEN CAMPAIGNS

Build once. Run forever.

→ Webhook in Clay
→ Scraper watches job boards 24/7
→ New matches push into Clay
→ Clay enriches, cleans, scores
→ Leads go straight into sequences

6️⃣ TECH STACK

What we actually use:

Lead sourcing: Apollo, Clay, LeadMagic
Sending: Instantly.ai, Smartlead, lemlist
CRM + automation: Attio, n8n

Total: around $1,500–3,000 per month depending on team size.

The new playbook:
→ Source from 5+ places
→ Use triggers, not random targeting
→ Enrich and score in Clay
→ Write relevant copy
→ Run evergreen campaigns
→ Track and optimize weekly

This system took us from $0 to $6M ARR.
