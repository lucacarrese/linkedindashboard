## HOOK FORMULAS

Every post opens with one of these. Pick the one that fits the content. Never start with "I".

### 1. Milestone + Result
```
[We/ColdIQ] [did X] at [$Y ARR]. [N months/years] later, [we hit $Z].
Here's the exact [system/process/playbook] that got us there:
```
*Example: "I joined ColdIQ at $2M ARR. 15 months later we hit $7M. Here's the exact growth system that got us here:"*

### 2. Stat Escalation
```
In [year], [metric] was [number].
In [year], that number hit [number].
By [year], it crossed [number].
```
*Example: "In 2023, the B2B average was about 120 cold emails to get one positive reply. In 2024, that number hit 200. By 2025, it crossed 430."*

### 3. Data Analysis
```
We analyzed [N] [posts/campaigns/clients].
[N small] of them drove [massive result].
Here's what they had in common:
```
*Example: "We analyzed 220 posts across the ColdIQ team. 5 of them drove 11,300+ interactions. Here's what those 5 had in common:"*

### 4. External Thesis
```
[Company/person] just [launched/said/explained] [X].
Here's what it is, why it matters, and how to [apply it]:
```
*Example: "Anthropic just launched Claude Dispatch and it will kill OpenClaw. Here's what it is, why it matters, and how to set it up in 5 minutes:"*

### 5. Step-by-Step Promise
```
Step-by-step how to [outcome]:
(adding [$X / N result] along the way)
```
*Example: "Step-by-step how to generate 20M+ impressions per year on LinkedIn: (adding $5,000,000 in new ARR along the way)"*

### 6. ONE Thing Hook
```
[Credibility claim with number].
The top performers all did ONE thing first.
```
*Example: "I built outbound systems for 30+ teams. The top performers all did ONE thing first."*

### 7. Specific Dollar Hook
```
[This/My] [thing] generates $[X] in [timeframe].
Here are the exact [tools/steps] [we/I] use:
```
*Example: "This Tech Stack generates $500,000 in monthly pipeline. Here are the exact tools we run across 70+ clients at ColdIQ."*

### 8. Bold Industry Statement
```
[Trend] [reshaped/killed/changed] [domain] in [year].
What [took/cost] [X] now [takes/costs] [Y].
[Contrarian implication].
```
*Example: "AI reshaped the SDR role in 2026. What took hours now takes minutes. Did that make SDRs less valuable? I believe the opposite."*

### 9. Experience Claim + Promise
```
After [N years] of [scaling X to $Y], [this is everything / here are the N things] I learned about [topic].
```
*Example: "After 3 years of scaling ColdIQ to a $7M business, this is everything we learned about getting clients from LinkedIn."*

### 10. Question Bridge
```
[Two sentence credible setup about a topic].
But [key question]?
```
*Example: "Anthropic has two of the hottest agentic products right now. But when should you use Claude Code vs Cowork?"*

## BODY STRUCTURES

### A — Numbered Framework
Best for: systems, playbooks, step-by-step guides, principles.

```
1️⃣ [Bold Header]

[2-3 line explanation]
[optional sub-bullets with →]

2️⃣ [Bold Header]

[2-3 line explanation]
[optional sub-bullets with →]
```
*Used in: Posts 1, 3, 6, 14, 15*

### B — Phase Breakdown
Best for: processes that happen in sequence over time.

```
Phase 1: [Name] (Weeks X-Y)

→ [action]
→ [action]
→ [action]

Then.

Phase 2: [Name] (Weeks X-Y)

→ [action]
→ [action]

Finally.

Phase 3: [Name] (Weeks X-Y)
```
*Used in: Post 10*

### C — Old Way vs New Way
Best for: industry shifts, paradigm changes, contrarian takes.

```
The old way:

[A)] [step]
[B)] [step]

The new way:

[A)] [step]
[B)] [step]

[What this means / implication]
[Two paths forward]
[Synthesis: the best teams combine both]
```
*Used in: Post 13*

### D — A vs B Breakdown
Best for: comparing two tools, approaches, or roles.

```
[Option A] → [Short tagline]

What it can do:
→ [capability]
→ [capability]

What it can't do:
→ [limitation]

Who it's for: [audience]

---

[Option B] → [Short tagline]

[same structure]

[One-line synthesis comparing both]
```
*Used in: Post 8*

### E — External Thesis → Internal Application
Best for: reacting to news, frameworks from others, industry ideas.

```
[Quote or describe external thesis]

We took that exact playbook into [domain]:

→ Not [X]. [What we actually do].
→ Not [Y]. [What we actually do].

[Our result with specific number]

[Implication for the reader — question or reframe]
```
*Used in: Post 4*

### F — Diagnostic Framework
Best for: helping reader identify their own problem.

```
Most [people/teams] think [common wrong assumption].

[Reframe: the real question is X]

[Category 1]
→ [diagnostic question]
→ [diagnostic question]

[Category 2]
→ [diagnostic question]

[Rule or action tied to their answer]

Start with one.
```
*Used in: Post 9*

## CTA TYPES

End every post with exactly one of these. Match to the post's goal.

### 1. Keyword Comment CTA
*Goal: generate comments, collect leads, share a resource*
```
Comment "[keyword]" and I'll [send you / share / drop] [specific resource] 👇
```
*Examples: "Comment 'Lead' so I can send you the infographic in full-res." / "Comment 'Claude Code for Ads' and I'll share the full skills repo with you 👇"*

### 2. Reflection Question CTA
*Goal: drive discussion, invite personal experience*
```
[Specific question about their situation / metric / approach]?
```
*Examples: "How many emails do you need to send to get a positive reply?" / "What's one principle you would add?" / "What's the split on your content?"*

### 3. Opinion Question CTA
*Goal: debate, preferences, prediction*
```
Which [option] fits your [context]?
What's the first [X] you'd [do]?
```
*Examples: "Which one fits your day-to-day?" / "What's the first skill you'd run from your phone? 👇"*

### 4. Resource Teaser CTA
*Goal: drive comment engagement to unlock content*
```
I [mapped / built / dropped] [specific thing] in the comments.
```
*Examples: "I mapped 20 concrete agent builds across Sales, Marketing, RevOps, and CS into one cheat sheet." / "I dropped the full breakdown in the comments."*

## VOICE RULES

### Numbers — always specific, never vague
- Never: "millions in revenue" → Always: "$7M ARR"
- Never: "hundreds of campaigns" → Always: "320+ campaigns"
- Never: "a lot of meetings" → Always: "1,100 meetings from content alone last year"

### Sentence rhythm — short, short, medium
- Short sentences (3-7 words) carry weight.
- One-line breaks for dramatic effect: "Then.", "Start with one.", "That's it.", "Nothing is random."
- Never two long sentences back to back.

### Lists — always → never bullets
- Main points: numbered (1️⃣ or 1 -)
- Sub-points within a section: →
- Sub-sub-points: ↳

### Credibility — woven in, never announced
- Don't say "I'm an expert in X"
- Say: "across 320+ campaigns", "for 70+ clients", "after 3 years of scaling to $7M"

### Avoid these words and patterns
- "Unlock", "leverage", "synergy", "game-changer", "revolutionize"
- "In today's world", "In the age of AI", "Let me tell you"
- Starting a sentence with "So" or "Well"
- Filler transitions: "Additionally", "Furthermore", "In conclusion"

### Structural markers that work
- Parenthetical impact additions: "(adding $5,000,000 in new ARR along the way)"
- "P.S:" for bonus tips or a second question
- "Here's the thing:" to introduce a contrarian point
- "Then." / "Finally." as single-line section separators in phase breakdowns
- Tool names dropped casually to establish technical credibility (Clay, Apollo, Instantly, etc.)

### Perspective — teach by showing, not theorizing
- Don't say "you should use signals" — say "we routed every engagement through Clay, scored it, and actioned it"
- Specific > general, always
- Show the system, not the concept
